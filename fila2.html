Descripción del producto

Siéntete cómodo sin sacrificar el rendimiento y la protección en tu día de trabajo con las zapatillas de seguridad antideslizantes Fila Memory Workshift. Donde Premium cumple con la utilidad, el Memory Workshift está fabricado con cuero duradero y superposiciones sintéticas para darte un zapato de trabajo fiable que satisface tus necesidades laborales.

Nuestros zapatos Fila Memory Workshift cuentan con una suela de goma antideslizante para proporcionar tracción contra superficies resbaladizas o húmedas. Aunque está diseñado para ayudar a prevenir resbalones, siempre debes tener precaución en superficies resbaladizas.
